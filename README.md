# PolicyDrift Lab

PolicyDrift Lab detects dangerous policy changes before agentic workflows are deployed.

## How to Run (from zero)
### Prerequisites
- Git
- Any modern browser

### Setup
1. `git clone https://github.com/sundaiclaw/sundai-policydrift-lab.git`
2. `cd sundai-policydrift-lab`

### Run
1. Open `index.html` directly in your browser
   - or host locally with: `python3 -m http.server 8080`
2. Visit `http://localhost:8080` (if using local server)
